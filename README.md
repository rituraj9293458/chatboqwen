# ChatboQwen

## About the Project

ChatboQwen is a full-stack AI chatbot application that I'm building to learn and experiment with modern AI technologies and backend development. The project combines a React frontend with a Node.js and Express backend, MongoDB for storing conversations, and LangChain with Ollama for integrating local large language models.

The goal is not only to build a chatbot, but also to understand how production-ready AI applications are designed and developed. This repository will continue to evolve as I learn new technologies and add more advanced features.

---

## Current Features

* User signup and login
* AI-powered chat interface
* Chat history stored in MongoDB
* LangChain integration for prompt handling
* Local LLM support through Ollama
* Modular backend built with Express.js
* React-based frontend

## Getting Started

Clone the repository:

```bash
git clone https://github.com/rituraj9293458/chatboqwen.git
```

### Frontend

```bash
cd chatbotqwen
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```
